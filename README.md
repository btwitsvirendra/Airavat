## 🚀 Quick Start - Two Ways to Run

### ⚡ Method 1: Simple (30 seconds)

**Mac/Linux:**
```bash
./run-simple.sh
```

**Windows:**
```bash
run-simple.bat
```

**Or manually:**
```bash
python -m http.server 8000
```

**Then open:** http://localhost:8000

---

### 🔥 Method 2: Professional with Vite (5 minutes)

```bash
# First time setup
npm install

# Start development server
npm run dev
```

**Opens automatically at:** http://localhost:3000

**Features:**
- ⚡ Lightning-fast hot reload
- 🎨 Modern development experience
- 🔧 Professional build tools
- 📦 Optimized for production

- **Architecture:** Class-based OOP pattern
- **Storage:** LocalStorage with 24h session

- **Styling:** Sass/SCSS support
